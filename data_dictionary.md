# ЁЯФа <samp>Data Dictionary</samp>

This document defines every field generated by the *glyph feature pipeline*, describing:
- what it represents  
- how itтАЩs computed  
- its data type  
- and possible values

### ЁЯУБ Top-Level Structure
Each processed glyph produces one entry inside `glyphs.catalog.json`:
```json
{
  "id": "",
  "filename": "",
  "glyph_url": "",
  "color": { ... },
  "metrics": { ... },
  "color_harmony": "",
  "mood": "",
  "created_at": { ... }
}
```

<br>

## ЁЯФ╣ Field Definitions

### id
- **Type:** string  
- **Example:** `"a3f92cd1"`  
- **Description:** A unique 8тАСcharacter hash assigned to the glyph.

---

### filename
- **Type:** string  
- **Example:** `"e3b45c_20250121_133402_af39dc12.png"`  
- **Description:** The output filename created based on dominant color + timestamp + unique ID.

---

### glyph_url
- **Type:** string  
- **Description:** Direct CDN url pointing to the stored glyph image.

---

## ЁЯОи Color Detection

### color.hex
- **Type:** hex string  
- **Example:** `"a1c2f3"`  
- **Description:** Dominant color of the glyph in hexadecimal representation (without `#` prefix)

---

### color.group
- **Type:** categorical  
- **Possible values:**  
`black`, `white`, `gray`, `gold`, `silver`,  
`brown`, `red`, `orange`, `yellow`,  
`green`, `blue`, `purple`, `pink`  
- **Description:** Semantic color group assigned based on HSV and luminance.

---

### color.rgb
- **Type:** list of 3 integers  
- **Example:** `[120, 180, 240]`  
- **Description:** Dominant color extracted using K-means clustering.

---

### color.lab
- **Type:** list of 3 floats  
- **Description:** Perceptually uniform color space for accurate color distance calculations

---

## ЁЯУК Metrics

### metrics.edge_density
- **Type:** float
- **Description:** Proportion of strong edges inside the non-transparent regions. Canny edge detection with thresholds (80, 160)

---

### metrics.entropy
- **Type:** float  
- **Description:** Shannon entropy grayscale pixel distribution (texture randomness).

---

### metrics.texture
- **Type:** float  
- **Description:** LBP (Local Binary Pattern) texture complexity measure.

---

### metrics.contrast
- **Type:** float (0тАУ1)  
- **Description:** Michelson contrast on masked luminance values.

---

### metrics.circularity
- **Type:** float  
- **Range:** 0тАУ1  
- **Description:** Shape compactness; 1 = perfect circle.

---

### metrics.aspect_ratio
- **Type:** float  
- **Description:** Width/height of the largest contour bounding box.

---

### metrics.edge_angle
- **Type:** float (degrees)  
- **Description:** Median orientation of strong edges (0тАУ180┬░).

---

## ЁЯМА Semantics+Mood

### color_harmony
- **Type:** categorical  
- **Values:**  
`analogous`, `complementary`, `none`  
- **Description:** Relationship between dominant and secondary colors.

---

### mood
- **Type:** categorical  
- **Values:**  
`minimalistic`, `futuristic`, `mysterious`, `energetic`, `organic`, `serene`  
- **Description:** HighтАСlevel classification based on combined metrics.

---

## ЁЯХТ Creation stamp

### created_at.date
- **Type:** string (YYYY-MM-DD)

### created_at.time
- **Type:** string (HH:MM:SS)
