# 🔱 Glyph Processor
Automatically process, analyze, and catalog visual glyphs with advanced metrics and mood classification.

This Python script allows you to:
- Upload images (glyphs) for processing.
- Detect dominant color and convert to RGB, HEX, LAB.
- Compute advanced image metrics:
  - Edge Density (structure complexity)
  - Entropy (information complexity)
  - Texture, Contrast, Circularity, Aspect Ratio, Edge Orientation
  - Color Harmony
- Assign one of 9 moods to each glyph:
serene, calm, energetic, chaotic, mysterious, futuristic, minimalistic, dramatic, playful
- Automatically generate metadata in JSON and JS formats.
- Save results locally as a ZIP or upload directly to GitHub.
- Incrementally update GitHub repository with new glyphs and metadata.

This creates a professional-grade glyph dataset suitable for web projects, galleries, or ML datasets.

## Features
- Automatic unique filenames using timestamps + UUIDs.
- Dominant color extraction with K-Means clustering.
- Robust mood classification based on multiple image features.
- JSON + JS export for easy integration into websites or apps.
- Incremental updates: preserves existing library when uploading to GitHub.

## How to Use
1. Run the script, and it will prompt you to select images for processing
2. Enter your GitHub username and repository name
3. Choose where to save results:
   1️⃣ Local ZIP: Saves processed images and metadata in a ZIP file.
   2️⃣ GitHub Upload: Uploads PNGs and metadata files (glyphs-data.json and glyphs.js) directly to the repo.

## Output
- Processed PNG files with unique filenames.
- JSON metadata: glyphs-data.json
- JS metadata: glyphs.js

## Notes
- Works best with high-quality PNG images with transparent backgrounds.
- Advanced metrics and mood detection are tuned for abstract, geometric, and glossy shapes.

# Mood Decision Rules

The script assigns one of nine moods (serene, calm, energetic, chaotic, mysterious, futuristic, minimalistic, dramatic, playful) based on a combination of image metrics and color features.

### Metrics
| Edge Density: Measures structural complexity. | High values → many edges/complex shapes.<br>Low values → smooth or minimal shapes. |
|---|---|
| Entropy: Measures informational complexity (how "busy" the image is). | High entropy → textured, detailed.<br>Low entropy → simple, uniform. |
| Brightness: Average RGB brightness. | Low brightness → dark, moody.<br>High brightness → vibrant, energetic. |
| Saturation: Intensity of dominant color. | High saturation → vivid, bold.<br>Low saturation → soft, calm. |
| Dominant Color Hue: Helps detect color temperature and style. | Warm colors (reds, oranges) → energetic, dramatic.<br>Cool colors (blues, purples) → calm, futuristic. |
| Color Harmony: Detects if colors follow complementary, analogous, or triadic schemes. |  |
| Shape Metrics: circularity, aspect ratio, edge angles can refine mood. |  |

### Mood Assignment Rules

Serene	Low entropy (<4), low edge density (<0.05), soft colors, low saturation
Calm	Low entropy (<5), moderate edge density (<0.08), pastel or cool hues
Energetic	High saturation (>0.45), warm colors, moderate complexity (entropy 4–6)
Chaotic	High entropy (>6), high edge density (>0.15), irregular shapes
Mysterious	Dark brightness (<90), cool colors (blue/purple), medium entropy (4–6)
Futuristic	Bright (>150), cool hue (blue/purple), low to moderate saturation (<0.35)
Minimalistic	Low entropy (<3.5), very low edge density (<0.03), simple shapes
Dramatic	High contrast, moderate to high entropy (5–7), bold warm colors
Playful	Medium entropy (4–6), bright colors, irregular or whimsical shapes

> Each metric is weighted according to visual importance, combining color, shape, and complexity for robust assignment.
